# VGDH Healthcare - Patient Portal 🏥

A modern, clean, and clinical web-based appointment system for **VGDH Healthcare**. This portal allows patients to request consultations across various departments and track their medical appointment history in real-time.

## ✨ Key Features
* **Department-Based Booking:** Patients can choose from Cardiology, Pediatrics, Dermatology, and more.
* **Real-time Status Tracking:** View if an appointment is `Pending`, `Approved`, or `Rejected`.
* **Medical History Log:** A card-based UI to review all past consultation requests.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.

## 🎨 Design System
* **Typography:** *DM Serif Display* (Headings) & *DM Sans* (Body).
* **Color Palette:** Healthcare Teal (`#0d9488`), Mint (`#ccfbf1`), and Clinical Cream (`#f0fdfa`).
* **Icons:** Bootstrap Icons.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom), Bootstrap 5.3.
* **Backend:** PHP (Procedural).
* **Database:** MySQL / MariaDB.
* **Deployment:** InfinityFree (Live).

## 🚀 Installation & Setup
1. Clone the repository: `git clone https://github.com/shainajaneee/VGDHHealthcareAppointment.git`
2. Import the `hospital.sql` file into your MySQL database.
3. Configure your database credentials in `db.php`.
4. Run via XAMPP/WAMP or upload the `htdocs` content to your host.
